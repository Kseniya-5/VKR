# Fashion CV Project

### Авторы: Балицкая Ксения, Герилович Илья

## Обзор
Этот репозиторий содержит реализацию проекта **Fashion CV** в рамках курсовой работы. Проект представляет собой модуль для работы с компьютерным зрением (Computer Vision) в сфере моды. Его главная задача — автоматическая обработка изображений одежды, извлечение характеристик (атрибутов) с помощью локальной нейросети и подготовка размеченного датасета для дальнейшего использования в моделях машинного обучения.



## Требования
- Python 3.10+
- Пакетный менеджер `uv` (рекомендуется) или `pip` / `poetry`
- Локально развернутая нейросеть (в данном проекте используется `llama3.2-vision:latest` через Ollama + Open WebUI)
- От 32 ГБ оперативной памяти и современный GPU (рекомендуется RTX 4080 или аналогичная) для быстрого инференса.
