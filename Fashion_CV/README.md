# Fashion CV Project

### Авторы: Балицкая Ксения, Герилович Илья

## Обзор
Этот репозиторий содержит реализацию проекта **Fashion CV** в рамках выпускной квалификационной работы (ВКР). Проект представляет собой модуль для работы с компьютерным зрением (Computer Vision) в сфере моды. Его главная задача — автоматическая обработка изображений одежды, извлечение характеристик (атрибутов) с помощью локальной нейросети, подготовка размеченного датасета и обучение Multi-Task Learning модели для системы рекомендаций.



## Требования
- Python 3.10+
- Пакетный менеджер `uv` 
- Локально развернутая нейросеть (в данном проекте используется `llama3.2-vision:latest` через Ollama + Open WebUI)
- От 32 ГБ оперативной памяти и современный GPU (рекомендуется RTX 4080 или аналогичная) для быстрого инференса.

## Пайплайн машинного обучения (`ml_pipeline`)

Архитектура ML-ядра разделена на 4 последовательных этапа:

1. **Salient Object Detection (Этап 1)**  
   Подготовка пользовательских фотографий: удаление "шумного" домашнего фона и выделение чистого объекта одежды для корректной работы нейросети.
2. **Dataset Preparation (Этап 2)**  
   Глубокая обработка сырых данных: парсинг вложенных JSON-аннотаций в плоский формат `pandas.DataFrame`. Включает Feature Engineering: группировку 76 типов одежды в 12 макро-классов, очистку цветов, сезонов и стилей от мусорных значений, а также кодирование признаков методами OHE и MultiLabelBinarizer. На выходе формируется компактный и плотный файл `processed_dataset.csv` (~62 000 строк, 43 колонки).
3. **Attribute Extraction (Этап 3)**  
   Создание Custom PyTorch Dataset и обучение единой сверточной/трансформерной нейросети (Multi-Task Learning) с несколькими "головами" для одновременного извлечения всех Fashion-атрибутов за один проход (inference).
4. **Fashion Ontology & RecSys (Этап 4)**  
   Написание алгоритмов рекомендаций. Система принимает извлеченные атрибуты вещи (например, `shirt dress`, `fall`, `casual`) и, используя экспертные правила колористики и онтологии моды, подбирает подходящую пару из виртуального гардероба пользователя.
