# Разработка контекстно-ориентированной системы подбора стиля с использованием многоуровневого анализа изображений и онтологии моды

Данный репозиторий содержит исходный код программного комплекса, разработанного в рамках дипломного проекта. 

Цель работы — создание интеллектуальной системы, которая анализирует изображения одежды пользователя с помощью методов компьютерного зрения (Computer Vision), сопоставляет их с правилами онтологии моды и предоставляет контекстно-ориентированные рекомендации по стилю через удобный пользовательский интерфейс.

Архитектура проекта разделена на две независимые, но логически связанные части: **ML-ядро (продуктовая часть)** и **Микросервисный backend (сервисная часть)**

## Структура репозитория

Проект состоит из двух основных модулей, каждый из которых имеет собственную документацию и среду выполнения:

### 1. [Fashion_CV](./Fashion_CV) (Продуктовая часть / AI)
Модуль для обучения и тестирования моделей компьютерного зрения.
* Многоуровневый анализ изображений (детекция, сегментация и классификация элементов одежды).
* Извлечение признаков (цвет, фасон, текстура).
* Интеграция с графом знаний (онтологией моды) для генерации рекомендаций.
* *[Перейти к документации ML-модуля ➔](./Fashion_CV/README.md)*

### 2. [Fashion_Bot](./Fashion_Bot) (Сервисная часть / Backend)
Пользовательский интерфейс и серверная инфраструктура.
* Telegram-бот для взаимодействия с пользователем (@FashionableSelectionBot).
* Асинхронная обработка тяжелых задач (инференс нейросетей) с использованием архитектуры Producer-Consumer (Celery + Redis).
* Сохранение статусов и истории в базу данных PostgreSQL.
* Полная контейнеризация и оркестрация (Docker, Kubernetes / Minikube).
* *[Перейти к документации backend-модуля ➔](./Fashion_Bot/README.md)*

## Стек технологий

**Computer Vision & Data Science:**
* Python 3.12
* PyTorch / OpenCV / Scikit-learn (или укажите ваши библиотеки, например Ultralytics/YOLO)
* Pandas / NumPy

**Backend & Инфраструктура:**
* Aiogram 3.x (Telegram Bot API)
* Celery & Redis (Очереди задач)
* PostgreSQL + SQLAlchemy/Alembic (База данных)
* aiohttp / Nginx (Веб-сервер и Reverse Proxy)

**DevOps:**
* Docker & Docker Compose
* Kubernetes (Minikube, StatefulSet, Jobs, ConfigMaps, Secrets)
* GitHub Container Registry (GHCR)
