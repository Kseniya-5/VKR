# Разработка контекстно-ориентированной системы подбора стиля с использованием многоуровневого анализа изображений и онтологии моды

Данный репозиторий содержит исходный код программного комплекса, разработанного в рамках выпускной квалификационной работы.

Цель работы — создание интеллектуальной системы, которая анализирует изображения одежды пользователя с помощью методов компьютерного зрения, извлекает ключевые визуальные признаки и предоставляет рекомендации по стилю через удобные пользовательские интерфейсы.

Проект разделён на две независимые, но логически связанные части: **исследовательский ML-контур** и **пользовательский сервис**. ML-контур отвечает за подготовку данных, обучение моделей и эксперименты с графовым представлением предметной области моды. Пользовательский сервис включает Telegram-бота, веб-интерфейс, backend API, базу данных и очередь фоновых задач.

## Структура репозитория

Проект состоит из двух основных модулей, каждый из которых имеет собственную документацию и среду выполнения.

### 1. [Fashion_CV](./Fashion_CV) — ML-контур / AI

Модуль для подготовки данных, обучения и тестирования моделей компьютерного зрения.

- автоматическая разметка изображений одежды;
- предобработка и очистка данных;
- извлечение признаков одежды: тип, цвет, стиль, сезонность;
- обучение и сравнение моделей ResNet50V1 и RandomForest;
- экспериментальная работа с графом знаний и онтологией моды.

[Перейти к документации ML-модуля ➔](./Fashion_CV/README.md)

### 2. [Fashion_Bot](./Fashion_Bot) — пользовательский сервис / Backend

Backend-инфраструктура, Telegram-бот и веб-интерфейс для работы пользователя с системой.

- Telegram-бот для взаимодействия с пользователем;
- веб-интерфейс для управления гардеробом;
- загрузка, просмотр и удаление фотографий;
- получение рекомендаций и сбор образов;
- асинхронная обработка ML-задач через Celery + Redis;
- хранение данных в PostgreSQL;
- контейнеризация и развёртывание через Docker и Kubernetes / Minikube.

[Перейти к документации backend-модуля ➔](./Fashion_Bot/README.md)

## Текущий статус

В рамках работы реализован пользовательский сервис с Telegram-ботом и веб-интерфейсом, а также исследовательский ML-контур для извлечения признаков одежды и проверки графового подхода к рекомендациям.
